## About Me

Blender x ComfyUI ecosystem developer, focused on bridging the gap between 3D creation and AI-powered workflows. I build tools that connect creative software with generative AI, making complex pipelines feel seamless.

- Building real-time bridges between Blender and ComfyUI
- AI-assisted 3D automation via MCP protocol
- End-to-end animation production workflows with ComfyUI nodes
- Growing the ComfyUI custom node ecosystem
